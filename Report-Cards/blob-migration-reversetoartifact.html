📦 Blob Migration Tool (Reverse)

Kanyadet Active Students Database — one-time run

What this does:
Reads every entry in studentPhotos and studentCerts. For each one:
1. Copies the blob (photoURL / birthCertBase64 + birthCertName) back into the matching student record
2. Removes the entry from the studentPhotos / studentCerts node

After migration, student records are restored to their original inline-blob format.
⚠️ Run once only. Keep this tab open until "Migration complete" appears. Do not use the main app while this is running.
